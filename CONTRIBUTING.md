# Contributing to LolScan

First of all, I would like to thank you for contributing!

## How to contribute?

### Reporting bugs
Before creating an issue with the bug, make sure it has:
- A clear title
- Steps on how to reproduce the problem
- Expected and actual result

### Code style
- Use lowerCamelCase for variables, private functions
- Use CamelCase for structs, functions

### Pull Requests
Before creating a pull request, make sure that:
- Your code compiles correctly
- Everything works properly

### Suggesting improvements
If you have a suggestion on how to improve the project:
1. Open an issue to discuss the changes
2. Clarify how exactly your suggestion solves the problem
